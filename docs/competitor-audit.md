# SP Automotive — Competitor Audit

Snapshot of how local Sarasota body shops and national/regional premium exotic specialists are presenting themselves online. Method: standard `WebFetch` of each site, no scraping, no screenshots. Captured on 2026-04-29.

The point of this doc is not to list every competitor — it is to surface the patterns we should adopt, the ones we should reject, and to give SP Automotive a clear picture of what "elite collision repair on the web" looks like today.

## 5. Cross-cutting patterns

**Patterns SP Automotive already wins on (don't lose them):**
- Owner-led narrative (Serge). Almost no competitor has a named, technical principal.
- Premium typography + spacing. Most competitors look like 2014.
- Cinematic hero video. Direct exotic specialists use static carousels at best.
- Forensic / "factory" voice. Competitors blur into "we're great" claims; ours is specific.
- Custom cursor + film grain + scroll-pinned process narrative. Nobody in this set is using motion this way.

**Patterns SP Automotive should adopt (highest leverage):**
1. **Certification / authorization badge wall.** I-CAR Gold Class, paint manufacturer (PPG, Glasurit, AkzoNobel), tooling (Car-O-Liner, Celette, Accudraft), ADAS calibration tools. → Add a `<CertificationsStrip />` to home + brand pages. Phase C addition.
2. **Equipment & booth specs on the process page evidence layer.** Already in plan as Phase B3. Reinforced — every direct competitor either claims certifications without showing them or shows tooling without context. We can do both.
3. **Lifetime workmanship guarantee** (or whatever warranty actually exists) as a visible badge. Already mentioned in JSON-LD plans for Phase C — surface it visually too.
4. **A second, lighter conversion path** distinct from the full contact form. Photo-first quick estimate ("Send 3 photos, get a callback in 2 hours"). Lower-friction than the full form. Out of current Phase A but a strong Phase C addition.
5. **SMS click-to-text** as a third conversion option alongside phone + form. One small `tel:` → `sms:` button near the phone CTA.
6. **Real review with attribution / link to source** when testimonials land — beats anonymous quotes for credibility.

**Patterns to explicitly reject:**
- Information-dense mega-menus (Foreign Affairs).
- Empty "5-star reviews" headers with no content (MasterClass, Lambo Miami, Luxury Custom Garage). We already gate this — confirmed correct call.
- Generic stock body-shop photography (B & B Camaro hero, Caliber corporate template). We should keep video-first and shop-evidence-first.
- Burying NAP on the contact page (B & B). Footer NAP rolling out in Phase A4 fixes our version of this.
- "FREE QUOTE!!!" energy. Stay confident in tone.

## 6. Plan adjustments suggested by this audit

These are not new commitments — they are observations that should inform Phases B and C as we get to them:

- **Phase B3 (Process evidence layer):** When Serge sends the photo library, prioritize equipment-and-tooling shots, not just intake/sign-off. Caliber and Gerber don't show tooling. Dimmitt mentions it in text. Showing it beats both.
- **Phase C (new pages):** Add a **`/certifications` or `/equipment` page** to the C list — this is the page that turns vague "factory-correct" claims into evidence. Aligns with the forensic voice.
- **Phase C (new pages):** Add a **lighter quote funnel** as `/estimate` — one phone, three photos, one description. Distinct from the full `/contact`. This was deferred under "conversion volume" in the plan but the pattern is so consistent across the comp set that it deserves promotion.
- **Phase A3 (analytics):** Add `sms_cta_click` to the event list when SMS button ships.

I'll fold these into the plan when we move into Phase B/C. No code changes today.
